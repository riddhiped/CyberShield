# Cyber Shield

A full-stack cybersecurity threat detection and monitoring platform built using React.js, Flask, and PostgreSQL.

## Features

* Real-time threat analysis
* SQL Injection detection
* XSS attack detection
* Malware activity detection
* Failed login monitoring
* Threat severity classification
* Interactive analytics dashboard
* CSV report export

## Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Recharts

### Backend

* Flask
* Python

### Database

* PostgreSQL

## Project Overview

Cyber Shield simulates a Security Operations Center (SOC) dashboard used by security analysts to monitor, classify, and visualize cyber threats. The platform assigns threat scores, categorizes attacks, and provides real-time analytics for security monitoring.
